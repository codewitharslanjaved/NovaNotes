# 🌌 NovaNotes — Galaxy-Themed Notes UI

A visually immersive **note-taking app** built with **Next.js + TypeScript + Tailwind CSS**, themed around cosmic nebula aesthetics for a calming journaling experience.

🌐 **Live Demo:** [novanotesweb.vercel.app](https://novanotesweb.vercel.app/)
**Source Code:** [NovaNotes GitHub Repo](https://github.com/codewitharslanjaved/NovaNotes)

---

## ✨ Features

* 🌌 **Galaxy & Nebula-inspired UI** — cosmic visuals, soothing gradients, dark mode
* ⏱️ **Focus Mode (25 min)** — stay productive with timed sessions (Pomodoro style)
* 🏆 **Achievements** — unlock milestones as you take notes & stay consistent
* 💾 **Auto-save** while typing — never lose your thoughts
* 🔒 **Local-first** storage — your notes remain private

## 🛠️ Tech Stack

* ⚛️ **Next.js (App Router)** + TypeScript
* 🎨 **Tailwind CSS** with custom cosmic theme
* 🧩 **UI Components**:  custom components
* ⚡ **Hooks** for editor state, auto-saving
* 📦 **Vercel** for seamless deployment

---

## 📂 Project Structure

```bash
├── app/              # Next.js pages & layouts
├── components/       # UI parts like editor, header, buttons
├── hooks/            # Auto-save, theme switch, focus timer logic
├── lib/              # Helpers, data formatting
├── public/           # Images, backgrounds
├── styles/           # Tailwind overrides & cosmic theme
├── components.json   # UI registry
├── next.config.mjs
├── tailwind.config.ts
├── tsconfig.json
├── package.json
└── README.md
```

## ⚙️ Getting Started

```bash
# Clone the repo
git clone https://github.com/codewitharslanjaved/NovaNotes.git
cd NovaNotes

# Install dependencies
npm install

# Launch dev server
npm run dev  # Go to http://localhost:3000

# Build for production
npm run build
npm start
```
